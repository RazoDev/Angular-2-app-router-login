
# Angular-2-app-router-login
app con angular 2.  Formulario de login con acceso admin


### Instalacion

```sh
$ cd /La ruta donde quieras clonar/
$ git clone https://github.com/RazoDev/Angular-2-app-router-login.git
$ cd path/carpeta clonada
$ npm install
$npm start
```
